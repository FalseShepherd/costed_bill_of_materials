Installation Instructions:

1) Install Python3.4 (32 bit version) from the official Python site. Add python
and pip to the system path as instructed below for Windows:

http://www.howtogeek.com/197947/how-to-install-python-on-windows/

2) Run "python one_time_setup.py"

3) After step 2 completes, run "python run_project.py"

4) Navigate to http://127.0.0.1:5000/ with a web browser to view the project
